# @marcuscastelo/ci-presets

Private workspace package for shared CI preset code.

## Status

This package currently exports an empty ESM module from `src/index.js`.

## Usage

```js
import "@marcuscastelo/ci-presets";
```

## Development

- Keep this package private unless publishing is explicitly required.
- Add CI preset exports in `src/index.js`.
- Run `pnpm test` from the repo root when behavior changes.
